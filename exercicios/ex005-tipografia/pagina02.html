Trabalhando com fontes

Mudando a fonte

Para mudar a fonte dos textos, pode-se utilizar um novo arquivo .css ou na tag <head> coloque dentro da tag <style>:

parte_que_quer_mudar{ font-family: fonte_desejada;}

Exemplo de código:

<style>

body{

font-family: Arial, Helvetica, sans-serif;

}

h1, h2{

font-family: 'Times New Roman', Times, serif;

}

</style>

Mudando o tamanho

Para mudar a medida das letras, pode-se utilizar um novo arquivo .css ou na tag <head> coloque dentro da tag <style>:

parte_que_quer_mudar{ font-size: tamanho_desejado;}

Obs:

As medidas de letras são divididas em medidas absolutas e medidas relativas:

  • Medidas absolutas:
    • cm
    • mm
    • in(polegadas)
    • px
    • pt
    • pc
  • Medidas relativas
    • em (1em equivale a 16px)
    • ex
    • rem
    • vw
    • vh
    • %

Exemplo de código:

<style>

body{

font-size: 1em;

}

</style>

Peso das fontes

Para mudar o peso das letras, pode-se utilizar um novo arquivo .css ou na tag <head> coloque dentro da tag <style>:

parte_que_quer_mudar{ font-weight: peso_desejado;}

Obs:

Para o font-weight existem os seguintes pesos:

  • Pesos com nomes:
    • lighter
    • normal
    • bold
    • bolder
  • Pesos numéricos:
    • 100 - 900

Exemplo de código:

<style>

body{

font-weight: normal;

}

</style>

Exemplo de código:

</style>

Shorthand font

O shorthand font é uma mãozinha que condensará todas as configurações de fonte em uma só.

Ordem dos font no shorthand:

font-style -> font-weight -> font-size -> font-family

Exemplo de código sem shorthand:

<style>

h1{

font-family: 'Work Sans', sans-serif;

font-weight: bolder;

font-size: 3em;

font-style: italic;

}

</style>

Exemplo de código com shorthand:

<style>

h1{

font: italic bolder 3em 'Work Sans', sans-serif;

}

</style>
