Para criar alinhamentos, utilizamos 'text-align: alinhamento_desejado' dentro da tag <style>

Exemplo de código:

<style>

body{

font: normal, 16px, Arial, Verdana, serif;

text-align: left;

}

h1{

text-align: center;

}

p{

text-align: justify;

text-indent: 20px;

}

</style>

Obs:

Esse 'text-indent' é o espaço que a primeira frase do parágrafo terá do lado esquerdo
