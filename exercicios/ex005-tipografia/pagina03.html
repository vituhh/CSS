Pegando fonte externa

Para pegar uma fonte no Google Fonts, basta ir no Google Fonts, pegar uma fonte, copiar o código fornecido e colar o código dentro da tag <style>

Exemplo de código utilizando a fonte 'Playfair Display':

<style>

@import url('https://fonts.googleapis.com/css2?family=Grape+Nuts&family=Playfair+Display:ital,wght@0,400;0,500;0,600;0,700;0,800;0,900;1,400;1,500;1,600;1,700;1,800;1,900&display=swap');

body

{

font-family: 'Playfair Display', Arial, Helvetica, sans-serif;

font-size: 16px;

color: black;

}

</style>
