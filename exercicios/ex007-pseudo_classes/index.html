Pseudo-classes

Pseudo-classes são palavras/comandos que definem estados de elementos dentro da HTML.

As pseudo-classes em CSS são representadas por ":"

Exemplo de pseudo-classe utilizando div:

Código:

<style>

Definindo as características das div's

div{

display: inline-block;

background-color: rgb(198, 194, 194);

border: 1px solid black;

text-align: center;

height: 200px;

width: 200px;

}

Definindo certo comportamento das div's utilizando a pseudo-classe 'hover'

div:hover{

background-color: yellow;

}

</style>
