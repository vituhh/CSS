Como criar as personalizações

ID

Para criar um seletor personalizado, basta criar um 'id' dentro da tag. Assim, quando for formatar o estilo, ao invés de colocar o tipo de elemento, coloque '#id_colocado{personalização}'.

Observações:

  • Em HTML é id = em CSS é #
  • Em HTML é class = em CSS é .
  • Não se pode utilizar o mesmo id para duas coisas no mesmo documento

Class

Para que se possa fazer alteração em vários elementos ao mesmo tempo, utiliza-se 'class="nome_da_classe"'. Conselho: não dê nome a classe de acordo com a forma do elemento, dê um nome relacionado a função.

Obs:

Quando for definir as configurações para um elemento que está com uma classe, não se coloca #nome_da_classe. Coloca-se .nome_da_classe

.Pode-se colocar um elemento em duas classes diferentes, por exemplo:

<h2 class="avancado destaque">CSS avançado</h2>

Com isso, esse H2 terá as características de ambas as classes.

.Também é possível colocar id e class em um mesmo elemento, por exemplo:

<h1 id="principal" class="avancado">Criando Sites com HTML e CSS</h1>

Entretando, as configurações do id se sobrepõem as configurações do class.

Span

A tag <span> serve para selecionar uma parte do texto e assim, consegue-se criar um id ou uma class para aquela parte do texto e alterar somente o local selecionado.
