Alinhando os elementos da tabela

Para alinhar elementos da tabela, utilizamos os seguintes comandos dentro do <style>:

  • text-align
    • Alinhamento horizontal
  • vertical-align
    • Alinhamento vertical

Exemplo de Código

Código HTML

<table>
...<tr> <
......<td>A1</td>
......<td>B1</td>
......<td>C1</td>
...</tr>
...<tr>
......<td>A2</td>
......<td>B2</td>
......<td>C2</td>
...</tr>
...<tr>
......<td>A3</td>
......<td>B3</td>
......<td>C3</td>
...</tr>
</table>

Código CSS

<style>
body{
font-family: Arial, Helvetica, sans-serif;
}
table{
width: 400px;
height: 400px;
border-collapse: collapse;
}
td{
border: 1px solid black;
padding: 8px;
text-align: center; /* Alinhamento horizontal */
vertical-align: middle; /* Alinhamento vertical */
}
</style>

Resultado:

A1 B1 C1
A2 B2 C2
A3 B3 C3
