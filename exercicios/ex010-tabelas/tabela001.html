Minha primeira tabela

Para criarmos uma tabela, primeiro temos que entender a hierarquia dos elementos da tabela.

HIERARQUIA DE TABELAS SIMPLES

TABLE = tabela
TABLE ROW = linha de tabela
TABLE HEADER = cabeçalho de tabela
TABLE DATA = dado de tabela

Esses são as representações dos elementos em tags:

  • TABLE: <table>
  • TABLE ROW: <tr>
  • TABLE HEADER: <th>
  • TABLE DATA: <td>

Código

Exemplo de código:

<table>
...<tr> <!-- Primeira linha -->
......<td>A1</td>
......<td>B1</td>
......<td>C1</td>
...</tr>
...<tr class="linha">
......<td>A2</td>
......<td>B2</td>
......<td>C2</td>
...</tr>
...<tr>
......<td>A3</td>
......<td class="dado">B3</td>
......<td>C3</td>
...</tr>
</table>

Exemplo de como alterar o estilo:

<style>
...body{
......font-family: Arial, Helvetica, sans-serif;
...}
...table{
......width: 400px;
......border-collapse: collapse; !!ESTE COMANDO É PARA JUNTAR AS BORDAS!!
...}
...tr.linha{
......background-color: lightgray;
...}
...td{
......border: 1px solid black;
......padding: 8px;
...}
...td.dado{
......background-color: rgb(252, 252, 76);
...}
</style>

Resultado:

A1 B1 C1
A2 B2 C2
A3 B3 C3
